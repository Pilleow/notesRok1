Powtórka C

Typy liczbowe

= 1 bajt (<0; 255> unsigned, <-128; 127> signed)

= 2 bajty (<0; 65635> unsigned, <-32768; 32767> signed)

= 4 bajty (; )

= 8 bajtów (; ) Dodatkowo kwalifikatory i wymuszają pewne wielkości bitowe.

Tekst jest przechowywany jako lista , np.: "Test" = { 'T', 'e', 's', 't', '\0' } gdzie '\0' to znak końca napisu. btw. '7' =/= "7"

Funkcje

void hello(char s) {
    printf("Hello, %d !", s);
}

Deklaracja stałych

`#define X N`
// makrodefinicja, kompilator pod każdy X podstawia N.
// Makrami można również zastąpić funkcje, np.:
`#define SQR(x) ((x) * (x))`

const int x = N;`
//zmienna, której wartość nie może ulec zmianie.

enum months = { JAN=1, FEB, MAR, APR, MAY, JUN, JUL, AUG, SEP, OCT, NOV, DEC }
// nadajemy pierwszej wartość, i potem każdej następnej automatycznie jest przypisana wartość o 1 większa od poprzedniej , np. MAY = 5.

int i, j, k;
double eps = 1.0e-6;
// każda zmienna musi być zadeklarowana przed użyciem.
{
    extern float eps;
    // ...
}
// EXTERN redeklaruje eps w nowym bloku. Używając EXTERN nigdy nie ustawiamy wartości.

static int = 5;
//Zmienna z kwalifikatorem STATIC pamięta wartość pomiędzy wywołanymi funkcjami.

Zasięg zmiennych

  • Zmienne lokalne są widoczne wewnątrz bloku ( {...} ), w którym zostały zadeklarowane, jak i w blokach w nim zagnieżdżonym.
  • Zmienne zewnętrzne istnieją cały czas, są definiowane poza wszystkimi blokami.
