Kodowanie ZNAK MODUŁ (ZM)

Najstarszy bit = Znak (1 = -; 0 = +) 0 = 0000 0000 -0 = 1111 1111 MIN = MAX = (gdzie n = liczba bitów)

Kodowanie ZNAK MODUŁ ODWROTNY (U1)

Dla liczb ujemnych odwracamy znaki, najstarszy bit def. znak, jak w ZM

Kodowanie UZUPEŁNIENIOWE (U2)

Dodajemy 1 do kodu U1, najstarszy bit def. znak, jak w ZM 0 = 0000 0000 MIN = 1000 0000 MAX = 0111 1111 ZAKRES =

Kodowanie ZMIENNOPRZECINKOWE

W skrócie notacja wykładnicza F = gdzie: m - mantysa, liczba stałoprzecinkowa, c - cecha, wykładnik potęgowy p - podstawa części potęgowej dla liczb binarnych p = 2

Słowo maszynowe: Z CCCC MMMM MMMM gdzie Z = znak, C = cecha, M = mantysa
